# Virtual-MIDI-CC64
In this project,We are trying python to create a virtual MIDI CC64 signal,It has a signal of 0-127 that can be switched.So you can use it to replace your sustain pedal.在这个项目中，我们用python来创建一个虚拟的MIDI CC64信号，具备0-127的信号可以切换。因此，它可以接受信号来替换或模拟电钢琴延音踏板。
提示：此脚本或许需要您安装rtmidi或mido的必备前置API组件，请下载Windows Visual构建工具并对C++构建器全局安装。本脚本后续必定解决这个麻烦的问题。
